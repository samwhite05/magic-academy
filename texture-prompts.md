# Magic Academy — Nano Banana Texture Prompts

All textures are 64x64 pixel art for use with Oraxen.

## Style Reference
> Include this at the start of each new chat window to anchor the style.

**Style:** 64x64 pixel art, Minecraft item aesthetic, dark fantasy / magic academy theme. Clean black outline, vibrant saturated colors, subtle shading with 2-3 tones per color, no anti-aliasing, white background. Consistent lighting from top-left.

## Notes

- UI items (`ui_spellbook_bg`, `ui_spellbook_divider`, `ui_hideout_bg`) don't need Nano Banana textures — handle these as flat colored images directly in Oraxen YAML.
- If Nano Banana drifts from the style mid-window, paste the Style Reference again to re-anchor it.
- Save outputs as PNG with transparency before importing into your Oraxen resource pack.
