# Dungeon World Template for wizard_ruins
# 
# This folder should contain a pre-built Minecraft world that will be copied
# when a new dungeon instance is created.
#
# To create a dungeon world:
# 1. Build your dungeon rooms in a separate Minecraft world
# 2. Copy the entire world folder here (region/, level.dat, etc.)
# 3. The instance manager will copy this to dungeon_<instanceId> on each run
#
# For now, a blank flat world will be generated automatically.

# Room locations (relative to world spawn):
# - Room 1 (Puzzle): x=-100, z=0
# - Room 2 (Mob Wave): x=-200, z=0
# - Room 3 (Miniboss): x=-300, z=0
# - Room 4 (Boss): x=-400, z=0
# - Room 5 (Treasure): x=-500, z=0
